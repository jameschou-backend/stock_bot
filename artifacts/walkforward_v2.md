### 原因分析

Strategy C 的 **Rank Drop 出場機制**在市場下跌時自動換到相對強勢的股票，不需要大盤過濾即可規避市場風險：

- 大盤跌時，持股的絕對排名快速下滑 → Rank Drop 自動觸發出場
- 進場條件要求進入 top 10%，市場空頭時幾乎無股票滿足 → 自然持現金
- 大盤過濾的設計哲學是「減少持股數」，但 Strategy C 已靠相對強弱實現同等效果

**結論：Strategy C 不需要大盤過濾，維持現行設定即可。**

## 整體結論

### 🟠 存疑（9/14 通過，不達可信門檻 11/14）

**正面信號：**
1. Folds 9-14（2022-2024）平均 Sharpe 2.230，高於 Folds 1-8 的 1.518——模型特徵在近期市場更有效
2. 2022 熊市（Fold 9/10）表現穩健（Sharpe 1.957 / 2.710），熊市防禦力已驗證
3. 大盤過濾無效反而是好消息：Rank Drop 機制已內建市場適應性
4. 失敗 Fold 多有合理解釋（熊市初期訓練不足、震盪市成本過高）

**負面信號：**
1. 2018 全年（Fold 1/2）連續失敗：訓練期缺乏足夠熊市樣本是系統性問題
2. 2023H2（Fold 12）：同年上下半年的天壤之別（6.39 → -0.72）顯示策略對市場風格轉換敏感
3. 高換手率在震盪市有成本風險（Fold 14：Cost Drag ~58%/yr）

## 附：大盤過濾對 Strategy C 的理論解釋

```
Strategy A（月頻）需要大盤過濾：
  → 月底換股，熊市中不得不持有一整月的下跌股
  → 大盤過濾可減少持股數，提早切到現金

Strategy C（日頻 Rank Drop）不需要大盤過濾：
  → 熊市中個股相對弱化 → 排名下滑 → Rank Drop 當日出場
  → 進場條件 top 10% 在熊市幾乎無股票滿足 → 自然持現金
  → 日頻換倉的本質是「隱性動態大盤過濾」
```
